2003, 9hells

Palito

A cute project from the college days, inspired on Total Annihilation, Palito (which means "Stick") is a Real-Time Strategy game with usability in mind. Pitty I could not find a stable version with the heightmap working, there were quite some beautiful maps on the style of TA.

  • Game makers:
    • Rodolfo "barrett" Borges (9hells): lead programmer
    • Carlo "zED" Caputo: network code and windows port
  • Platform: C, SDL
