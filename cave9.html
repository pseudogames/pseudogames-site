Cave9

A cave exploration game inspired on sfcave, the one-button-game. It started-up on a code marathon from 9hells.org.

  • Jam: 9hells' weekend marathon
  • Game makers:
    • Daniel "dbb" Borges (9hells): Cave shape prototype
    • Victor Bogado: Cave shape improvements
    • Ary Monteiro Jr.: Texture Art
    • Carlo "zED" Caputo (9hells): Core code
    • Rodolfo "barrett" Borges (9hells): Core code and releases
    • Jonas Kühner (criatipos): True-type Font
  • Platform: C, SDL, OpenGL
  • Links:
    • Source on Github
    • Download Source and Data files (1M)
    • Included on Linux Format Mag, issue 108 (2008-08)
  • Controls:
    • Up: Straight Up
    • Left, Right: Diagonal Up
    • Escape: Exit
