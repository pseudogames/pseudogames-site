Survivor

A survivor style game developed during SPJAM 2012 game marathon, São Paulo, Brazil. The first version of the game (check the Alpha video) was fully developed in just 48 hours. It was implemented using C as main language and SDL as graphic library, and in under 48 hour. Originally it was designed to be played as an arcade game with multiplayer support.

  • Jam: SPJAM 2012
  • Theme: cooperation
  • Game makers:
    • Carlo "zED" Caputo: programmer, level designer, game design
    • Pedro Mariano: programmer
    • Caires Vinicius: path finding
    • Pedro Jatoba: character art
  • Platform: C, SDL
