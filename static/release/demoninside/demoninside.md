Demon Inside
============

Unity 2D casual arcade made in 2 hours.

* date: 2015-07-31
* theme: igreja, casa, top, boi, iluminati, bomba
* team: LUA.NET
* duration: 2h
* platform: Unity 3D

CONTROL
-------
* A/Left, D/Right: Move
* Escape: Exit
